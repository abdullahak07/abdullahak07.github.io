Projects

1. Prototype for Integration Of Face Mask Detection and Person Identification Model

We merge two different models (Face Mask Detection and Person Identification Model) into one single algorithm, we used InsightFace model to detect the faces and recognize them. The dataset for face images for Face mask detection was taken from Kaggle.

2. Covid-19 Detection Using X-Ray and Machine Learning

In this project we use a Machine learning model embedded in spark framework to get the result of Covid-19 patients through X-ray.
The dataset for X-ray images was collected from Kaggle and Google Images.

3. Automatic Number Plate Detection System using Python and Machine Learning

The project focused on the Automatic Detection of Vehicle Number Plates using Machine Learning, which can be used
in traffic management applications like automatic gate control for authorized/non-authorized vehicles.


4. PDF Extractor using OCR in Python
In this project we use the OCR as well as Pytessaract to convert image into text file and save in the format of .doc,.txt.
Apart from this , its GUI interface also provides option to merge, split the PDF file.
